# DocFlow AI — Frontend

React 19 + Vite SPA for DocFlow AI.

**Status:** Scaffolded as a placeholder. The frontend is implemented starting at
**Module 1 · Task 7 (Frontend Auth)**, once the authentication API (Tasks 2–5)
is in place.

Planned stack (see [../docs/Architecture.md](../docs/Architecture.md)):
- React 19 + Vite
- Tailwind CSS + shadcn/ui
- React Router
- TanStack Query (server state) + Zustand (UI/global state)
- React Hook Form + Zod
- Axios (single instance with auth + refresh interceptors)

Design: enterprise, minimal, fast, light/dark — inspired by Stripe, Linear,
Notion, GitHub, and Vercel dashboards. Usability over animation.
